# Extension Icons

This directory must contain the following icon files before the extension can be loaded in Chrome:

- **icon16.png** -- 16x16 pixels (favicon, toolbar)
- **icon48.png** -- 48x48 pixels (extensions management page)
- **icon128.png** -- 128x128 pixels (Chrome Web Store, installation dialog)

All icons should be PNG format with a transparent background. Replace these placeholders with your actual brand icons.
